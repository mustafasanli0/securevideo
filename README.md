# Secure Video
Video Encryption Script


## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Requirements


```
1.) Install ffmpeg
```
```
2.) Install Nodejs
```


## Encrypt Video

```
./encrypt.sh {outputFolder} {videoUrl}
```
```
./encrypt.sh test test.mp4
```
